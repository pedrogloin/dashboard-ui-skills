# 📊 Dashboard UI Skills Stack

Stack de skills curadas e validadas para construção de UI de **Dashboard e Data Analytics** com agentes de IA (Claude Code, Cursor, Cline, etc.).

Fontes: [skills.sh](https://skills.sh), [GitHub](https://github.com), [ClawhHub](https://clawhub.ai), [AgentSkills](https://agentskills.so)

---

## ⚡ Instalação Rápida

```bash
# Instalar todas as skills do stack de uma vez
npx skills add vercel-labs/agent-skills/vercel-react-best-practices
npx skills add anthropics/skills/frontend-design
npx skills add vercel-labs/agent-skills/web-design-guidelines
npx skills add shadcn/ui/shadcn
npx skills add wshobson/agents/tailwind-design-system
npx skills add supabase/agent-skills/supabase-postgres-best-practices
npx skills add vercel-labs/next-skills/next-best-practices
npx skills add pbakaus/impeccable/polish
npx skills add pbakaus/impeccable/animate
npx skills add pbakaus/impeccable/frontend-design
npx skills add nextlevelbuilder/ui-ux-pro-max-skill/ui-ux-pro-max
npx skills add anthropics/skills/canvas-design
npx skills add coreyhaines31/marketingskills/analytics-tracking
npx skills add anthropics/skills/xlsx
```

## 🏆 Stack Recomendada (ordem de instalação)

```
1. vercel-react-best-practices        → Base do projeto React
2. shadcn                             → Componentes de UI
3. tailwind-design-system             → Sistema de design
4. frontend-design (anthropics)       → Padrões de design
5. canvas-design                      → Gráficos e visualização
6. supabase-postgres-best-practices   → Backend / queries
7. analytics-tracking                 → Métricas e eventos
8. polish                             → Refinamento visual final
9. animate                            → Animações
```
